# 🚀 DropZone – Cloud-Based File Storage System

DropZone is a modern **cloud-based file storage and management system** that allows users to securely upload, organize, and share files with ease. It provides a clean UI, efficient file handling, and powerful features like search, analytics, and folder management.

### 🌐 Live Demo
https://drop-zone-sandy.vercel.app/

---
## 📌 Features

- 📁 **File & Folder Management**
  - Upload files
  - Delete files
  - Create folders
  - Organize content efficiently

- ⭐ **Star System**
  - Mark important files/folders for quick access

- 🔗 **File Sharing**
  - Share files with other users securely

- 📤 **Drag & Drop Upload**
  - Easy file upload with intuitive drag-and-drop support

- 🔍 **Search & Filter**
  - Quickly find files using search and filtering options

- 📊 **Storage Usage Analytics**
  - Visualize storage usage and file distribution

- 📱 **Responsive Design**
  - Optimized for mobile, tablet, and desktop devices

## ⚙️ How It Works

1. Users authenticate using **Clerk**
2. Files are uploaded via **React frontend**
3. Backend processes requests using **Node.js & Express**
4. Files are stored securely in **Cloudinary**
5. Metadata is managed using **MongoDB**
6. Users can organize, search, star, and share files
